길이와 index 를 사용하는 자료형 Array

js는 java의 ArrayList가 없고 Array와 거의 동일한 자료가 있다.

Object 와 Array 의 차이

  • Array 는 자료의 순서인 index 와 길이가 무조건 존재한다.
  • Object 는 key 와 value 로 이루어진 자료 구조 (==java.Map)
  • Object 는 인덱스와 길이가 존재하지 않는다.
  • Array 와 index를 키로 사용하지만 물리적으로 존재하지 않는 순서다.
  • js 에는 key 가 순서가 있고 길이가 있는 Object 를 Array로 변환하는 함수가 있다.

배열의 필드 접근법

  • list.0 (x)
  • list[0] (o)
  • list.get(index); (X java 에서 가능)
